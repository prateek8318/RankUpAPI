# Content Microservice

This microservice handles:
- Qualifications management
- Exams management
- Home content for users

## Controllers
- `Controllers/QualificationsController.cs` - Qualifications endpoints
- `Controllers/ExamsController.cs` - Exams endpoints
- `Controllers/HomeContentController.cs` - Home content endpoints

## Services
- `Services/QualificationService` - Qualification operations
- `Services/ExamService` - Exam operations
- `Services/HomeContentService` - Home content operations
